What you need:

- 1 ổ USB (format: FAT hoặc FAT32), đương nhiên là còn đủ chỗ để chứa file save 
- 1 đĩa CD-R trắng
- PS2SaveBuilder (trong file memcardsave.zip mình đính kèm)

Progress

A. Tạo đĩa boot LaunchEFL
1.Giải nén file DumpBios-USB.zip, bạn đc DumpBios-USB.iso
2.Dùng 1 chương trình ghi đĩa nào đó để burn file iso trên ra CD-R

B. Copy file save vào ổ USB
1.Giải nén file memcardsave.zip, chạy ps2save-builder.exe trong thư mục Ps2_Save_Builder
2.Click open, chọn đến file save của bạn (chỉ đc chọn 1 thôi đấy)

3.Tạo một folder mới với tên là Root/ID của file save, ở đây là BASLUS-20312FF090600
4.Chọn tất cả các file trong cột File Name~>phải chuột chọn Extract~>chọn đến thư mục tạo ở trên rồi Save
5.Làm tương tự với các file save khác
6.Copy tất cả các folders đã tạo ở bước B4 và B5 vào ổ USB

C. Transfer from USB to PS2 Memcard
1.Cắm ổ USB vào PS2
2.Boot PS2 bằng đĩa đã tạo ở bước A2
từ đây mình dùng screen shot của Pcsx2 (1 emu ps2) để hướng dẫn còn bạn phải làm trên máy PS2, hoàn toàn tương tự thôi



Ấn Select
chọn Default>Misc>FileBrowser(ấn O để chon)



chọn OK> chọn Default: File Browserchọn Default: File Browser



chọn Mass:/ (chính là ổ USB của bạn)



chọn file save bạn muốn copy, ấn R1, chọn copy (nếu bạn muốn chọn nhiều file cùng một lúc thì trước khi ấn R1, bạn ấn X, những file hiện dấu hoa thị phía trước tức là đã đc chọn)

ấn nút tam giác, nếu bạn muốn copy file save vào thẻ nhơ thì chọn Mc0:/ hoặc Mc1:/ (nếu chỉ có một thẻ nhớ thì màn hình chỉ hiện Mc0:/) , còn muốn copy vào ổ cứng thì chọn hdd0:/, ở đây mình chọn copy vào thẻ nhớ thứ 2 (Mc:1/)



ấn R1, chọn Paste và chờ một lúc ~ > DONE
